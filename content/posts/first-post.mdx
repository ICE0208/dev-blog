# 블로그 제작 계기

개발 관련 글을 작성하기 위해 블로그를 만들었다.

첫 번째 이유는 여러 블로그 플랫폼 서비스를 사용해보았지만, 만족스럽지 않았기 때문이다. 두 번째 이유는 직접 블로그를 만들면 좀 더 꾸준히 글을 작성할 수 있을 거라는 기대가 있었기 때문이다.

<br />

# 블로그 제작 과정

블로그 제작을 결심했을 때 이미 다른 사이드 프로젝트를 진행 중이었기에 시간이 많지는 않았다.  
그래서 유튜브 강의를 보며 따라 만드는 방식으로 진행했다.

[참고한 유튜브 강의](https://youtu.be/tSI98g3PDyE?si=srnMEz83Ic7bjhPe)

<br />

# 블로그 제작 후기

shadcn ui를 사용해보고 싶었는데, 블로그를 제작하면서 간단히나마 경험해볼 수 있어서 좋았다.

또한 velite라는 도구를 사용해서 mdx를 콘텐츠로 변환했는데, 생각보다 만족스러웠다.

<br />

# 앞으로...

유튜브 강의를 보고 거의 99% 똑같이 따라 만든 블로그지만, 기능을 하나하나씩 추가해보려고 한다.

생각한 기능들...

- 댓글
- 포스트 목차
- 포스트 검색
- 포스트 카테고리

<br />

# 여담

도메인도 구매해서 붙였다.  
도메인 값이 그렇게 크지는 않지만, 돈이 들어간 블로그라고 생각하면 더 꾸준히 글을 쓸 수 있을 거라 생각한다.
